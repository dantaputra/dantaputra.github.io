Projects

Search And Rescue (SAR) Robot

This robot based on EILERO (SAR robot from PENS) which I developed by adding computer vision features and a 4 DOF (Degree Of Freedom) robotic arm.

Uses 3 types of controllers: Arduino Pro Mini for angle reading via IMU sensor, STM32F4 which access other sensor and control robot actuator, and Jetson Nano to detect objects via camera.

I created autonomous systems algorithm for robot actions, and also object detection algorithm using YOLO (CNN object detection) that uses a model trained with custom objects + image sharpening algorithm (gaussian filter + unsharp masking) to detect and identify custom object for the robot competition.

Paper : https://ieeexplore.ieee.org/document/10052838
